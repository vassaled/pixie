# Pixie (v0.1.2)

Pixie is a browser-based pixel art and sprite editor.

## Features

* Pixel drawing tools:
  * Pencil, eraser, fill
  * Line, rectangle, circle (filled and stroke)
  * Color picker
* Color system:
  * Hex input
  * Palette support
* Layer system:
  * Multiple layers
  * Visibility toggle
  * Opacity control
* Animation system:
  * Frame-based workflow
  * Frame duplication and deletion
  * Onion skinning
  * FPS control
  * Playback system
* Selection tool with move support
* Undo and redo system
* Grid toggle and zoom controls
* Draggable floating panels (tools, colors, layers, animation)
* Context menu actions
* Export options:
  * PNG (single frame)
  * Sprite sheet
* Project save/load (.json)
